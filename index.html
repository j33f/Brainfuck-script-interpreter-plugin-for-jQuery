An HTML5 page running Brainfuck ?! Yes, you can !

This is a demonstration of the power of Brainfuck with HTML5.

Brainfuck is a powerful programming language and come with great things :

  • Only 8 characters to know !
  • US ASCII only : stop get brainfucked with all those stupid unicode stuffs
  • No human can read it clearly : code obfuscation is bundeled directly in Brainfuck !
  • Easy to compress : the 8 characters are repeated all along the program, that is very compressing friendly
  • It's enlarge your p3n1s ! (studies are in progress to prove that)
  • It's not a patented technology !

By using the Brainfuck jQuery plugin, you will be able to use simething like : 

<script language="brainfuck">
  ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
</script>

to make a «Hello World!»
